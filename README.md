# SAM AI Technologies — Task 1: Web Scraping

## Overview
Web scraping project using Python (BeautifulSoup + Requests).
Scrapes book data from books.toscrape.com.

## Files
- `web_scraper.py` — Main scraping script
- `books_dataset.csv` — Output dataset

## How to Run
pip install requests beautifulsoup4
python web_scraper.py

## Skills Used
- HTML parsing with BeautifulSoup
- Pagination handling
- CSV dataset creation
